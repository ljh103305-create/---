# 예금 이자 계산 웹앱 – Product Requirements Document (PRD)

## 1. 프로젝트 개요
**HTML, CSS, JavaScript**를 사용해 백엔드 없이 로컬스토리지에 데이터를 저장하는 **예금 이자 계산 웹앱**.
사용자가 입력한 **원금, 이자율, 기간** 값을 기반으로 **만기 시 수령액과 총 이자액**을 계산한다. **단리/복리 선택 기능**과 **이전 계산 내역 관리(localStorage)** 기능을 포함한다.

---

## 2. 과제 목표
| 구분 | 목표 |
|------|------|
| 기능적 | 예금 이자 계산 및 내역 관리 |
| 비기능적 | 직관적이고 접근성 높은 UI/UX |
| 기술적 | HTML/CSS/JS + LocalStorage, 백엔드 없음 |

---

## 3. 기능 상세

### (1) 입력 및 계산
- 입력 항목: 원금(₩), 연이자율(%), 기간(년), 이자 방식(단리/복리)
- [계산] 클릭 시 결과 표시: 만기금액, 총이자액
- 복리 선택 시 연단위 복리 계산 적용

### (2) 내역 저장 및 관리
- localStorage에 계산 결과 저장
- 내역 항목: 입력값, 계산 결과, 계산일
- 기능: 불러오기 / 삭제 / 수정 / 전체 초기화

### (3) 정렬 및 필터
- 기간순, 이자율순, 계산일순 정렬
- 복리/단리별 필터

### (4) 알림 기능 (선택사항)
- Notification API로 만기 알림 발송

## 7. 계산 로직
- **단리(Simple Interest)**  
  `이자 = 원금 × (이자율 ÷ 100) × 기간`  
  `총액 = 원금 + 이자`

- **복리(Compound Interest)**  
  `총액 = 원금 × (1 + 이자율 ÷ 100) ^ 기간`  
  `이자 = 총액 - 원금`

- 소수점 처리: 원 단위 반올림

---

## 8. 예외 처리
- 입력값 비어 있음 / 숫자 아님 → 경고창
- 음수, 0 입력 시 계산 불가
- localStorage 용량 초과 시 경고
