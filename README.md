# 💰 예금 이자 계산기

간편한 예금 이자 계산 웹 애플리케이션입니다. 원금, 이자율, 기간을 입력하면 단리/복리 방식으로 만기 금액과 이자액을 계산하고, 모든 계산 내역을 로컬에 저장하여 관리할 수 있습니다.

## ✨ 주요 기능

### 📊 계산 기능
- **단리 계산**: 원금 × 이자율 × 기간
- **복리 계산**: 원금 × (1 + 이자율)^기간
- 실시간 만기 금액 및 이자액 표시
- 원화 포맷팅 자동 적용
- **그래프 시각화**: Chart.js를 활용한 이자 성장 그래프

### 💾 내역 관리
- LocalStorage를 통한 자동 저장
- 계산 내역 불러오기/수정/삭제
- **계산 이름 지정**: 각 계산에 이름 부여 가능
- **이름 변경**: 저장된 계산의 이름 수정 가능
- 전체 초기화 기능

### 🔍 필터 및 정렬
- **정렬**: 계산일순, 기간순, 이자율순, 만기금액순
- **필터**: 단리/복리별 필터링
- 실시간 내역 업데이트

### 🚨 입력 검증 및 알림
- 빈 값 및 잘못된 입력 실시간 검증
- 커스텀 알림창 (5초 자동 사라짐)
- 포커스 자동 이동 기능

### 🎨 사용자 경험
- 반응형 디자인 (모바일/태블릿/데스크톱 대응)
- 직관적인 UI/UX
- 부드러운 애니메이션 효과
- 접근성 향상 (키보드 입력, 라벨 지원)

## 🚀 실행 방법

### 1. 파일 다운로드
프로젝트 파일을 다운로드하거나 클론합니다.

```bash
# Git을 사용하는 경우
git clone <repository-url>
cd deposit-interest-calculator
```

### 2. 웹 브라우저에서 실행
`index.html` 파일을 더블클릭하거나 웹 브라우저로 열기

```bash
# 또는 로컬 서버 실행 (Python 3)
python -m http.server 8000

# 그 다음 브라우저에서 http://localhost:8000 접속
```

### 3. 바로 사용하기
1. 원금, 연이자율, 기간을 입력
2. 단리 또는 복리 선택
3. "계산하기" 버튼 클릭
4. 결과 확인 및 자동 저장

## 💡 계산 공식

### 단리 (Simple Interest)
```
이자 = 원금 × (이자율 ÷ 100) × 기간
만기 금액 = 원금 + 이자
```

### 복리 (Compound Interest)
```
만기 금액 = 원금 × (1 + 이자율 ÷ 100)^기간
이자 = 만기 금액 - 원금
```

**소수점 처리**: 원 단위로 반올림

## 📊 데이터 구조

각 계산 내역은 다음과 같은 구조로 저장됩니다:

```javascript
{
  id: "2025-01-15T10:30:00.000Z",      // 고유 ID (ISO 문자열)
  name: "2년 정기예금",                 // 계산 이름 (선택사항)
  principal: 1000000,                   // 원금
  rate: 3.5,                            // 연이자율 (%)
  years: 2,                             // 기간 (년)
  type: "simple" | "compound",          // 이자 방식
  interest: 71400,                      // 이자액
  totalAmount: 1071400,                 // 만기 금액
  createdAt: "2025-01-15"               // 계산일
}
```
